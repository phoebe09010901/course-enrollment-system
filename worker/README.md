## 預期責任

Worker 應負責：

- 執行耗時或可重試的背景任務。
- 處理 AI 生成、整理、驗證或批次工作。
- 回寫任務狀態與結構化輸出。
- 避免阻塞 LINE webhook、admin 或前台互動流程。

Worker 不應負責：

- 呈現前台頁面。
- 直接處理 LINE webhook 的即時回應。
- 承擔 admin UI。
- 定義品牌 style 或模板本身。

## 初始任務流程

每個任務建議遵循：

1. 建立任務。
2. 驗證輸入。
3. 讀取必要的 style、template 或 skill 規格。
4. 執行背景處理。
5. 驗證輸出。
6. 儲存結果與狀態。

更多規格見 `../docs/AI_WORKER_WORKFLOW.md`。
