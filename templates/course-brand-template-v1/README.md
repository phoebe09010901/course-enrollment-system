# course-brand-template-v1

This is the first implemented front-end course recruitment landing page template.

## Scope

- HTML structure
- CSS implementation
- Responsive behavior
- Hero composition
- Section composition
- Front-end animation and interaction

## Out of Scope

- SQL
- Webhook
- Backend services
- Form schema
- Payment flow

## Files

- `index.html`: Static landing page markup.
- `css/page.css`: Template styling that consumes `styles/tokens/course-brand.css`.
- `js/page.js`: Front-end interactions only.
- `assets/course-workspace.svg`: Local visual asset for the hero workspace.

## Design Foundation

The template follows:

- `styles/course-brand-template-v1.json`
- `styles/layout-rules/landing-page.md`
- `styles/typography/course-brand.md`
- `styles/motion/animation-style.md`
- `styles/tokens/course-brand.css`
