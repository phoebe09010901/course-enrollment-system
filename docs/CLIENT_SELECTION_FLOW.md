# Client Selection Flow

## 文件目的

本文件定義客戶收到三款招生頁預覽後的 LINE 選版流程、提醒流程與逾期規則。內容屬於流程規格，目前尚未有 LINE webhook、資料庫、worker 或 admin 實作。

## 流程總覽

```text
三款預覽完成
  -> 系統產生三個 preview_url
  -> LINE 通知客戶
  -> 等待客戶選擇 A / B / C
    -> 客戶選擇：記錄 selected_proposal_id，進入正式招生頁預覽製作
    -> 期限前一天未選：LINE 自動提醒
    -> 三天內未選：預覽網址作廢，標記 expired
```

## 狀態定義

| 狀態 | 說明 | 下一步 |
| --- | --- | --- |
| `proposal_ready` | 三款預覽已完成，尚未通知 | 發送 LINE 通知 |
| `proposal_sent` | 已通知客戶，等待選款 | 等待回覆或排程提醒 |
| `proposal_reminded` | 期限前一天已提醒 | 等待回覆或到期作廢 |
| `proposal_selected` | 客戶已選擇 A / B / C | 進入正式招生頁預覽製作 |
| `proposal_expired` | 三天期限已過且未選擇 | 視需求重新開啟提案 |

## 選款解析

LINE AI 應能辨識以下客戶回覆：

| 客戶回覆範例 | selected_proposal_id |
| --- | --- |
| 我要 A 款 | `proposal_a` |
| 選 A | `proposal_a` |
| A 款 | `proposal_a` |
| 我要 B 款 | `proposal_b` |
| 選 B | `proposal_b` |
| B 款 | `proposal_b` |
| 我要 C 款 | `proposal_c` |
| 選 C | `proposal_c` |
| C 款 | `proposal_c` |

若客戶同時提到多款，例如「A 跟 C 都不錯」，AI 不應直接記錄選擇，需請客戶明確回覆其中一款。

## 選版通知話術

您好，我們已依照您的課程需求，完成 3 款招生頁視覺提案。

請於 3 天內選擇其中一款作為後續製作方向：

A 款｜清爽藝術風  
預覽網址：  
{{proposal_a_preview_url}}

B 款｜高級品牌風  
預覽網址：  
{{proposal_b_preview_url}}

C 款｜雜誌作品集風  
預覽網址：  
{{proposal_c_preview_url}}

請直接回覆：

- 我要 A 款
- 我要 B 款
- 我要 C 款

提醒您：預覽連結將於 {{expires_at}} 失效。若超過期限未選擇，需重新開啟提案流程。

## 逾期前一天提醒話術

提醒您，3 款招生頁預覽連結將於明天失效。

您可以直接回覆：

- 我要 A 款
- 我要 B 款
- 我要 C 款

如果您還不確定哪一款比較適合，也可以直接問我：「可以幫我比較 A、B、C 嗎？」我可以依照您的課程受眾、品牌調性與招生目標，協助您做選擇。

## 客戶選款後確認話術

好的，已為您記錄選擇的樣板方向：{{selected_label}}｜{{selected_template_name}}。

接下來我們會依照您選定的樣板，製作正式招生頁預覽。完成後會再透過 LINE 通知您確認。

系統將同步記錄：

```json
{
  "selected_proposal_id": "{{selected_proposal_id}}",
  "selected_at": "{{selected_at}}"
}
```

## 多款或不明確選擇話術

我目前還不能確定您最後想選哪一款。

請您直接回覆其中一個選項：

- 我要 A 款
- 我要 B 款
- 我要 C 款

我收到明確選擇後，才會幫您送出並進入正式招生頁預覽製作。

## 正式頁完成通知話術

您好，已依照您選定的 {{selected_label}}｜{{selected_template_name}} 完成正式招生頁預覽。

請您點選以下網址確認：

正式招生頁預覽：  
{{official_preview_url}}

確認時可以直接回覆：

- 內容確認
- 需要修改

如果需要修改，也可以直接告訴我想調整的地方，例如課程資訊、段落順序、照片、文案或按鈕文字。

## 試營運中通知話術

目前此流程為免費試營運階段，主要目的是協助建立課程招生頁並優化流程。

後續若有正式收費方案，會再另外公告或說明。

## 預覽網址作廢後話術

您好，這次的 3 款招生頁預覽已超過選擇期限，預覽連結目前已失效。

如果您仍想繼續進行，我們可以重新開啟提案流程，並再次確認課程需求、素材狀態與製作方向。

您可以直接回覆「重新提案」，我會協助您重新整理需求。
