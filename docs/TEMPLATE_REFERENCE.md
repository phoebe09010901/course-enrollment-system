# Template Reference System

## 文件目的

本文件是課程招生頁的「版型參考資料庫」。這 10 份網站風格分析報告不是一般靈感參考，而是未來產生招生頁版型時的正式依據。

Chat B 或 Codex 在產生、重構或調整招生頁前，必須先從本文件指定：

- `template_id`：主要版型依據。
- `secondary_template_id`：可選，作為輔助風格或局部節奏參考。

未指定 `template_id` 時，不應直接產生新版型。

## 必填欄位

每一筆 template reference 必須包含：

- `template_id`
- `report_file`
- `website_name`
- `source_url`
- `suitable_course_types`
- `suitable_industries`
- `visual_keywords`
- `color_palette`
- `hero_type`
- `layout_features`
- `typography_rules`
- `button_style`
- `image_strategy`
- `animation_style`
- `preferred_usage`
- `avoid_rules`
- `notes_from_user`

## 使用規則

- 產生招生頁前，先宣告「本頁採用 `TPL-xxx` 作為主要版型依據」。
- 可補充一個 `secondary_template_id`，但不得混用過多版型。
- 版型參考需遵守 `docs/STYLE_SYSTEM.md` 的全域規則。
- 不得把原網站照抄成另一個模板；應萃取 layout、節奏、比例、圖片策略與品牌感。
- 若 template 的原始網站有英文裝飾大字，必須移除或轉化為中文短句 / 抽象圖形。
- 若 template 的桌機字級偏大，必須依 `STYLE_SYSTEM.md` 下修。
- 不新增獨立 Contact / 聯絡我們 / 快速諮詢區塊。

- `suitable_industries`: 畫室、藝術工作室、個人品牌、招生頁
- `visual_keywords`: 乾淨、簡單、清楚、小工作室感、留白型、不擁擠、生活感
- `color_palette`: 淺色底、自然生活色、低飽和輔色、溫和對比
- `hero_type`: 左文右圖，圖片大，主標清楚但不巨大
- `layout_features`: 區塊清楚、留白充足、4:5 圖片、左文右圖、資訊層級簡單
- `typography_rules`: 字體不要太大；以大小、粗細、留白建立層級；桌機標題需克制
- `button_style`: 大圓角實心按鈕，穩定、親近、不要過度科技感
- `image_strategy`: 生活感圖片、工作室情境、手作過程、4:5 比例為主
- `animation_style`: 輕量淡入、低干擾，重點在閱讀節奏
- `preferred_usage`: 作為手作、繪畫、花藝、小型工作室招生頁的主要版型依據
- `avoid_rules`: 不要出現英文裝飾大字；不要把桌機標題做太大；不要做得擁擠
- `notes_from_user`: 可學習簡約、字體大小粗細排版、生活感圖片、4:5 圖片、大圓角實心按鈕
